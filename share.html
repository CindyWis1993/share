CSS&SVG-轨迹动画

前端开发中经常需要实现各种动画效果，特别是移动动画。
这次就选择最常见的轨迹动画跟大家分享

1、直线轨迹动画

直线轨迹动画是轨迹动画中最为常用的一种，通常可以直接两点定线来实现；如果需要做速度的调整，则直接改变运动的时间即可： 

                
                    .straightLine {
                        animation: straightLine 2s alternate infinite;
                    }
                    @keyframes straightLine {
                        0% { transform: translate(0, 0); }
                        30% { transform: translate(100px, 0); }
                        60% { transform: translate(200px, 100px); }
                        100% { transform: translate(300px, 100px); }
                    }
(0, 0) (100, 0) (200, 100) (300, 100)

2、曲线轨迹动画

曲线轨迹动画多用于实现球类的轨迹效果、loading效果等，实现方法多种多样，可以直接使用rotate、也可以用纵向横向两个方向的运动结合实现效果；



rotate实现轨迹动画，可以利用容器嵌套，外层容器旋转实现，但只适用于轨迹为圆形或圆形的一部分的情况：

                
                    .rotateBox-1 {
                        animation: rotateLine1 2s linear infinite;
                    }
                    .rotateBox-2 {
                        animation: rotateLine2 2s linear infinite;
                    }
                    @keyframes rotateLine1 {
                        0% { transform: rotate(0deg); }
                        100% { transform: rotate(360deg); }
                    }
                        
                    @keyframes rotateLine2 {
                        0% { transform: rotate(135deg); }
                        50% { transform: rotate(315deg); }
                        100% { transform: rotate(135deg); }
                    }

rotateBox-2


两个方向运动结合，受限于一个平面运动的物体永远是受X轴以及Y轴的力控制，因此单独对一个物体无法进行改变，只能达到直线轨迹的效果，要想实现双向运动实现曲线轨迹，只能再增加一个元素，使其互相影响，一方负责X轴运动，另一方负责Y轴运动实现可以实现椭圆等特殊的轨迹效果，但是弧度的计算比较复杂，较难调整：


具体如何实现呢，首先为了创建一条不成直线的路径，我们希望X轴和Y轴的运动速度不同步，因此需要对X轴跟Y轴设置两种计时函数，所以，分别对X轴跟Y轴使用ease-in、ease-out

                
                    .moveLine-1 {
                        animation: moveLineX 2.5s infinite ease-in;
                    }
                    .moveLine-2 {
                        animation: moveLineY 2.5s infinite ease-out;
                    }
                    .moveLine-3 {
                        animation: moveLineX 2.5s infinite ease-in;
                    }
                    .moveLine-3::after {
                        animation: moveLineY 2.5s infinite ease-out;
                    }
                    @keyframes moveLineX {
                      50% {
                        animation-timing-function: ease-in;
                        transform: translateX(100px);
                      }
                    }
                    @keyframes moveLineY {
                      50% {
                        animation-timing-function: ease-out;
                        transform: translateY(100px);
                      }
                    }

直接用默认的ease-in&ease-out实现的效果就是普通的圆弧，一些特殊弧度的改变则需要调整贝塞尔曲线的值
CSS3动画速度的控制通过三次贝塞尔曲线函数实现，定义规则为:
cubic-bezier (x1, y1, x2, y2)

P0：默认值 (0, 0)
P1：动态取值 (x1, y1)
P2：动态取值 (x2, y2)
P3：默认值 (1, 1)

关于三次贝塞尔曲线最直接的理解是，将以一条直线放在范围只有 1 的坐标轴中，并从中间拿出两个点来拉扯（X 轴的取值区间是 [0, 1]，Y 轴任意），最后形成的曲线就是动画的速度曲线。

但是贝塞尔曲线实际应用于动画开发中时，由于计算比较复杂（具体的计算涉及到线性代数的矩阵相关知识，了解更多->图形算法：贝塞尔曲线），一般很难完全还原本身设计的动画轨迹，因此可以借助一些辅助工具实现： 点击就送可视化贝塞尔曲线
